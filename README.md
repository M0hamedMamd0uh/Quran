A simple and clean Quran web application built with Next.js. It allows users to browse Surahs and view Ayahs using a public Quran API.

---

## Features

- Browse all Surahs of the Quran.
- View Ayahs (verses) with translation.
- Clean and user-friendly interface.
- Built with Next.js for fast and SEO-friendly performance.
- Uses a public Quran API to fetch data dynamically.

---

## Technologies Used

- Next.js
- JS
- TypeScript
- Bootstrap
- SASS
- Public Quran API [https://mp3quran.net/api/v3/languages]
